1. Which, if any, of the following 3 code fragments are equivalent? Explain why they are different, if they are. Explain why they can have different parameters and be equivalent, if they are equivalent. 

            //code fragment 1
            $("li").each(function(idx, e) {
            $(e).css(“color”, “yellow”); });

            //code fragment 2
            $("li").each(function() {
            $(this).css(“color”, “yellow”); });

            //code fragment 3
            $("li").each(function(idx) {
            $(this).css(“color”, “yellow”); });
Answer: All of the code fragments are equivalent. In first fragment there are two parameters the first is index position of element and the second is current element. So the css is applied to element using passes selector(i.e "e"). In the second code fragments the parameters is not passed so css is applied to current selector(li) using "this" keyword. So there is no need to pass element in function.In the third code fragment there index is passed as parameter but "this" kyword is uses to apply css for current selctor.

2. Write a jQuery expression to find all divs on a page that include an unordered list in them, and make their text color be blue. 

            <div> no ul here </div>
            <div>
            This does contain a ul.
            <ul>
            <li> the first item </li>
            <li> the second item </li>
            </ul>
            </div>
            <script>
            <!--INSERT YOUR JQUERY CODE HERE -->
        </script>
    </body>
Answer: 
            $(document).ready($(function(){
                $("div").has("ul").css("color","blue");
                });

3. Write jQuery code to append the following div element (and all of its contents) dynamically to the body element.

<div><h1>JQuery Core</h1></div>

HTML:
